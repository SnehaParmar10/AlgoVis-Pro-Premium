# 📊 AlgoVis Pro | Algorithm Visualizer

An interactive Data Structures and Algorithms (DSA) visualizer that brings sorting logic to life with real-time animations.

## ✨ Features
- **Visual Sorting**: Watch Bubble Sort in action with color-coded comparison and swapping.
- **Dynamic Array Generation**: Generate random datasets of varying sizes.
- **Speed Control**: Adjust the visualization speed to understand step-by-step logic.
- **Neon UI**: Modern dark theme for better focus on algorithm execution.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Flexbox/Gradients).
- **Core Logic**: JavaScript (Async/Await, Promises for animation timing).

## 🧠 Why I built this?
To master the visualization of time complexity. Understanding algorithms is one thing, but seeing how iterations and swaps happen in memory (via visual bars) is crucial for a deep understanding of Computer Science fundamentals.
